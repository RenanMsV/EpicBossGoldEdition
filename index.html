All Potion Types

Here is a list of all the potion types that exist in Minecraft. You will mainly need this list to set correct potion type on some skills. All Potion Types SPEED Increases movement speed. SLOW Decreases movement speed. FAST_DIGGING Increases dig speed. SLOW_DIGGING Decreases dig speed. INCREASE_DAMAGE Increases damage dealt. HEAL Heals an entity. HARM Hurts an entity. JUMP Increases jump height. CONFUSION Warps vision on the client. REGENERATION Regenerates health. DAMAGE_RESISTANCE Decreases damage dealt to an entity. FIRE_RESISTANCE Stops fire damage. WATER_BREATHING Allows breathing underwater. INVISIBILITY Grants invisibility. BLINDNESS Blinds an entity. NIGHT_VISION Allows an entity to see in the dark. HUNGER Increases hunger. WEAKNESS Decreases damage dealt by an entity. POISON Deals damage to an entity over time. WITHER Deals damage to an entity over time and gives the health to the shooter. HEALTH_BOOST Increases the maximum health of an entity. ABSORPTION Increases the maximum health of an entity with health that cannot be regenerated, but is refilled every 30 seconds. SATURATION Increases the food level of an entity each tick.

How to Setup Locations

Youtube Tutorial You can save locations and spawn the boss there instead of spawning the boss with xyz coordinates. You can also use locations to setup timers. To save a location type /eb loc add [name] To delete a location type /eb loc remove [name] To list a location type /eb loc list To teleport to a location type /eb loc teleport [name] To spawn the boss at the location type /eb mob spawn [boss_id_name] [amount] [loc_name]

How to Setup Timers

Youtube Tutorial You can use timers to spawn a boss every N seconds/hours at one location. First you need to save a location. Example: file > plugins\EpicBoss-Updated-Edition\Timers\TimerExample.yml timer1: # timer id/name Bosses: testboss # bosses to spawn at that location. you can set it just 'test'. so it will spawn randomly every boss with the 'test' word in his name MaxAmount: 3 # max bosses amount at the same time RespawnTime: 120 # time to respawn the boss in seconds WalkDistance: 0 # max distance from the location that the boss can spawn timer2: # you can have more timers in the same file Bosses: testbossminion MaxAmount: 1 RespawnTime: 600 WalkDistance: 20

How to Setup Random Spawning

Youtube Tutorial You can spawn any boss randomly at any world and any biome. Example: file > plugins\EpicBoss-Updated-Edition\RandomSpawning\SpawningExample.yml spawning1: # random spawning id/name Mobname: testboss # boss name/id from MobsExample.yml Worlds: world # world name. you can set more using world,world_nether,world_the_end Biome: plains,desert,nether # biomes list, if blank the boss will spawn in every biome Chance: 0.01 # 1% chance for the mob to spawn (0 = 0%, 1 = 100%) Priority: 1 # priority over others random spawnings spawning2: # you can have more random spawning in the same file Mobname: testbossminion Worlds: world Chance: 0.1 # 10% chance for the mob to spawn. Priority: 1
